What's included

Everything you need
to validate your tags

TagSentry Lite runs as a persistent side panel — capturing, matching, and validating network requests against a configurable vendor library as you browse.

📡
Live Tab
Real-time network event feed for the active tab. Each row shows the request method, URL, matched vendor, and PASS / FAIL / WARN / EXCLUDED status. Click any row to expand the full URL, matched rule results with actual vs. expected values, query parameters, and POST body.
PASS / FAIL / WARN Filter by vendor Known vendors only
🏷️
Vendors Tab
50+ built-in vendor definitions covering analytics, advertising, and data platforms. Create custom vendors or override built-ins. Organize with color-coded Labels. Group by Category or Label. Import and export with conflict resolution — overwrite, add as copy, or skip.
50+ built-in Labels Import / Export
▶️
Flows Tab
Record multi-step user journeys — clicks, inputs, scrolls, and navigation — then replay them against any URL. View step-by-step status and a post-replay summary report. Export flows as JSON or CSV.
Record & Replay JSON / CSV export Step delay control
🍪
Cookie Manager
Define and apply browser cookies with full control over name, value, domain, path, SameSite policy, expiration, Secure, and HttpOnly flags. Cookies matching the active tab's hostname are auto-applied on navigation.
Auto-apply SameSite / Secure HttpOnly
↔️
URL Rewrites
Intercept and redirect outbound requests before they leave the browser using plain string or RE2 regex patterns. Toggle rules on and off instantly without a page reload. Active rewrites display a visible warning banner.
String or Regex Toggle instantly RE2 syntax
✉️
Email Generator
Generate deterministic test email addresses from a configured prefix and domain. Useful for repeatable QA flows that require consistent but unique addresses across multiple test runs.
Deterministic Configurable prefix
🔑
Token Config
Store a reusable token value — API key, auth token, or any string — for injection into flows and replay steps. Keeps credentials out of your recorded actions.
Flow-injectable Single stored value

Settings & Backup
Light/dark theme, font size (S/M/L), vendor grouping, Known Vendors Only filter, Clear on Navigate, and replay defaults. Export or restore a full backup — all settings, vendors, labels, cookies, rewrite rules, and tokens — as a single JSON file.

Quick start

Up and running
in five steps

01
Open the side panel
Click the TagSentry Lite icon in Chrome's toolbar to open the side panel on any tab.
02
Enable vendors
Go to the Vendors tab and enable the vendors you want to monitor. Built-in vendors are disabled by default.
03
Browse and inspect
Switch to the Live tab — requests appear as you browse. Click any row to expand rule-level results with actual vs. expected values.
04
Record a flow
Go to the Flows tab, click Record, interact with the page, then click Stop and save the flow.
05
Replay and validate
Open a saved flow, click Replay, and the extension steps through each recorded action while capturing the resulting network events.
Per-tab monitoring
Only the active tab is monitored. Switching tabs pauses capture for the previous tab, keeping memory usage bounded. Each tab maintains its own independent event history.
Vendor rule evaluation
Exclusion rules short-circuit first — a matching exclusion marks the event EXCLUDED before inclusion rules run. Required rules fail → FAIL; optional rules fail → WARN.

How to install
01
Download the zip
Click the download button above to save tsl-v1.7.0.zip to your computer.
02
Unzip the file
Extract the archive to a permanent folder — Chrome will need to access this folder while the extension is active.
03
Open Chrome Extensions
Navigate to chrome://extensions in your browser and enable Developer mode using the toggle in the top-right corner.
04
Load the extension
Click Load unpacked and select the folder you extracted in step 2. TagSentry Lite will appear in your extensions list.
05
Pin and start inspecting
Click the puzzle icon in Chrome's toolbar and pin TagSentry Lite for easy access. Navigate to any page to start capturing tags.
